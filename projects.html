Rent Management System

The rent management system service is a new feature in our smart world, designed using Python where owners can put their properties on rent and all the rent related information will be recorded into the system which can be visible by the respective client with proper authorization. The system can manage hundreds of owners and tenants with proper security. Services offered by the system include locality service information like electrician, plumber, carpenters, gas-delivery, drinking water supplier, milk supplier and newspaper supplier.

Rock Paper and Scissor

The Stone-Paper-Scissors game is a classic interactive web application developed using HTML, CSS, and JavaScript. Players can make choices by clicking on stone, paper, or scissors icons, and the computer generates its choice randomly. CSS styling provides an engaging visual interface, while JavaScript handles the game's logic, determining the winner based on the chosen items. This project demonstrates the combination of front-end technologies to create an enjoyable and interactive gaming experience that's easy to understand and play.
